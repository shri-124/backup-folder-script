Backup & Compress Folder Script

CLI tool that zips a folder, timestamps it, puts it in backups/, and supports optional --encrypt (GPG) and --cloud (simulated upload).

Quick Install

macOS/Linux/WSL:

curl -fsSL https://raw.githubusercontent.com/shri-124/backup-folder-script/main/install.sh | bash

Usage

backup-folder [OPTIONS] <folder>

Options:
  --encrypt                 Encrypt archive with a passphrase (GPG symmetric)
  --encrypt-recipient=<email>  Encrypt with recipient's public key
  --cloud[=github|s3]       Simulate upload to a provider
  --outdir=<path>           Custom backup directory (default: ./backups)

Examples:
  backup-folder ~/demo
  backup-folder ~/demo --encrypt
  backup-folder ~/demo --cloud=github
  backup-folder ~/demo --outdir=/tmp/my_backups

Requirements

  • Bash (macOS/Linux/WSL or Git Bash)
  • tar, gzip (preinstalled on macOS/Linux)
  • Optional: gpg for --encrypt

Uninstall

rm -f ~/.local/bin/backup-folder ~/bin/backup-folder /usr/local/bin/backup-folder
